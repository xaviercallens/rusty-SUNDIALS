## 1. Introduction
**Rusty-SUNDIALS** is a pure Rust, production-quality implementation of the CVODE solver from the SUNDIALS suite (originally developed by Lawrence Livermore National Laboratory). It is designed to solve initial value problems for ordinary differential equation (ODE) systems, featuring state-of-the-art numerical methods for both stiff and non-stiff problems.

## 2. Numerical Methods & Scientific Underpinnings

### Adaptive Multistep Methods
Rusty-SUNDIALS employs adaptive linear multistep methods using a **Nordsieck history array** representation. 
* **Non-Stiff Systems**: Handled via implicit Adams-Moulton formulas (orders 1–12).
* **Stiff Systems**: Handled via Backward Differentiation Formulas (BDF) (orders 1–5).

### Implicit Newton-Raphson Iteration
Unlike simple explicit predictor-corrector solvers, the BDF method in Rusty-SUNDIALS incorporates a rigorous implicit Newton-Raphson iteration loop to solve the non-linear algebraic system at each step:
$$ M \Delta = \left( I - \gamma J \right) \Delta = b $$
Where:
* $J$ is the Jacobian matrix evaluated via finite-difference approximation.
* $\gamma$ is a scalar dependent on the step size $h$ and the leading Nordsieck coefficient $l_0$.
* $M$ is factorized using LU decomposition (`sundials_dense` getrf/getrs routines) to compute the correction vector $\Delta$.

This mathematical architecture ensures that the stability region is bounded by the implicit formulation rather than the truncation limits of explicit functional iterations, enabling robust integration of highly stiff systems.

## 3. Scientific Validation Benchmarks
The solver has been autonomously validated against classic stiff use-cases from the scientific literature:

* **Oregonator (Belousov-Zhabotinsky Reaction)**: Developed by Field & Noyes (1974), this highly stiff system of 3 coupled ODEs features reaction rates spanning several orders of magnitude. It serves as the primary "standard candle" for numerical stiff solvers.
* **1D Brusselator**: A theoretical model for an autocatalytic oscillating chemical reaction with diffusion (Prigogine & Lefever 1968). By applying the Method of Lines over a spatial grid of 100 points, it yields a challenging sparse 200-ODE system.
* **Van der Pol Oscillator**: An oscillator with non-linear damping. When $\mu \ge 10$, the system exhibits extreme stiffness. Rusty-SUNDIALS successfully tracks the stiff limit cycle utilizing the dense implicit solver over millions of highly accurate adaptive steps.
